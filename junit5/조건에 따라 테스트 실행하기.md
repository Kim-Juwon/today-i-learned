# 조건에 따라 테스트 실행하기
어떠한 조건을 만족해야 테스트를 실행하려 할때 사용할 수 있는 기능들에 대해 알아본다.

대표적인 기능인 **환경 변수**, **OS(운영체제)**, **JRE** 에 따른 테스트 실행 여부 판별에 대해 알아본다.

## 종류
- org.junit.jupiter.api.Assumptions.*
	- assumeTrue(조건)
	- assumingThat(조건, 테스트)

- @Enabled, @Disabled
	- OnOS: 특정 OS
	- OnJre: 특정 JRE 버전
	- IfSystemProperty
	- IfEnvironmentVariable
	- If
